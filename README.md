# CVE Tracker

**What it does?**

CVE Tracker is a Python application that fetches vulnerability data from the NVD JSON API and extracts key CVE details such as ID, description, severity, and exploitability score.

Built as a learning project to practice API consumption, defensive JSON parsing, and clean code structure for security-related data.

# Features
* Fetch CVEs from NVD API

* CVSS v3.1 priority with v2 fallback

* Safe handling of missing metrics

* Object-oriented CVE model

# How to run
* Python version 3.13.7

* python main.py
